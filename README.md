VisionX-AI 🚀

AI-powered image analysis web application built using AWS Rekognition, FastAPI, HTML, CSS, and JavaScript.

---

✨ Features

- 👤 Face Detection
- 📦 Object Detection
- 📝 OCR / Text Detection
- ⭐ Celebrity Recognition
- 🛡 Moderation Analysis
- 🖼 Image Preview
- ⏳ Loading Animation
- ❌ Error Handling
- 📱 Responsive Modern UI

---

🛠 Tech Stack

Frontend

- HTML
- CSS
- JavaScript

Backend

- FastAPI
- Python

Cloud & AI

- AWS Rekognition

---

📂 Project Architecture

Frontend → FastAPI Backend → AWS Rekognition → AI Analysis → Frontend Results

⚡ How to Run

1️⃣ Clone Repository

2️⃣ Install Requirements

3️⃣ Run Backend

uvicorn main:app --reload

4️⃣ Open Frontend

Run index.html using Live Server.
